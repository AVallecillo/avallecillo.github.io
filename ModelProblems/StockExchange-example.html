Design considerations and requirements

The system will offer an initial (home) page with an introductory text, and the list of current stock quotes for a given set of stocks in a fixed Stock Exchange Market.

The system is an example of a transient application, so initially the list will be empty.

  • The page will provide one input field to allow the user to provide the name of the stock to be included in the list of monitored stocks, and submit such input by using a navigation mechanism (submit button). If the stock exists in the Stock Market, it will be added to the list and its quote showed. Otherwise the application will display an error message in a different page.
  • Another button in the home page will allow to update the list being displayed, with the current quotes for the selected stocks.
  • Finally, a third button will instruct the application to automatically update the list every certain period of time. The duration (in seconds) of such a period will be provided using an input field.
Stock quotes will be obtained by the application by invoking the appropriate external Web services. (Free web services for getting the latest stock quotes from several Stock Markets are available, e.g., at http://www.xmethods.net/ )
