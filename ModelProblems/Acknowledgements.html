Model Problems in Model-Driven Web Engineering -- Acknowledgements

This project has been development under the auspicies of the MDWEnet initiative, and partly supported by project TIN2005-25886-E.

This is an ongoing project, for which your inputs and comments are welcome. More importantly, we invite both new model problems and new solutions (using, e.g., different modeling approaches, notations, or MDWE proposals), as well as refinements and/or improvements to the existing ones.

The idea of using model examples for a given discipline is not new. We have borrowed it from the original idea by the Composable Systems Group at the Carnegie Mellon University, that built a list of candidate model problems for software architecture.

Finally, the concepts, ideas, views, analyses and opinions expressed in these pages are the authors' solely responsibility.
